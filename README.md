# PetCare

PetCare is an Android application designed to help users manage tasks related to pet care. The app allows users to add tasks, receive notifications, and track pet-related responsibilities.

## Features

● Add pet care tasks

● Feeding & vaccination reminders     

● Mark tasks complete

● Daily checklist
